# Comandos para Previsualizar Logos ASCII de Neofetch

Este repositorio contiene una lista de comandos para previsualizar fácilmente los diferentes logos ASCII de distribuciones Linux (y otros sistemas) que vienen incorporados en la popular herramienta de información del sistema `neofetch`.

## ¿Para qué sirve?

Neofetch puede mostrar un logo ASCII diferente usando la opción `--ascii_distro <nombre>`. Esta lista te permite probar rápidamente muchos de los logos disponibles sin tener que editar tu archivo de configuración cada vez. Encuentra el logo que más te guste y luego configúralo de forma permanente.

## Modo de Uso

1.  Busca en la lista de abajo el nombre de la distribución (`<nombre>`) cuyo logo quieras previsualizar.
2.  Copia las **dos líneas** correspondientes a esa distribución (la línea `echo` y la línea `neofetch`).
3.  Pega esas dos líneas en tu terminal y presiona Enter.
4.  Neofetch se ejecutará mostrando el logo de esa distribución específica.
5.  Repite con otros nombres hasta encontrar tu favorito.

**Nota:** La lista se generó a partir de los comentarios en el archivo de configuración por defecto de Neofetch. Es posible que algunos nombres no funcionen si tu versión de Neofetch no los reconoce o si la interpretación del nombre no fue exacta.

## Configuración Permanente

Una vez que hayas encontrado un logo que te guste usando los comandos de arriba, puedes configurarlo permanentemente en Neofetch siguiendo estos pasos:

1.  Edita tu archivo de configuración (puedes usar `nano` o `vim`):
    ```bash
    nano ~/.config/neofetch/config.conf 
    ```
    *(O `vim ~/.config/neofetch/config.conf`)*

2.  Asegúrate de que la línea `image_backend` esté descomentada (sin `#` al inicio) y diga:
    ```bash
    image_backend="ascii"
    ```

3.  Busca la línea `image_source=`, descoméntala si es necesario, y cambia el valor entre comillas por el **nombre exacto** de la distribución que elegiste (normalmente en minúsculas):
    ```bash
    # Ejemplo si elegiste EndeavourOS:
    image_source="endeavouros" 
    ```

4.  Guarda los cambios en el archivo:
    * En `nano`: `Ctrl+O`, presiona `Enter`, luego `Ctrl+X`.
    * En `vim`: Presiona `Esc`, escribe `:wq` y presiona `Enter`.

5.  ¡Listo! La próxima vez que ejecutes `neofetch` en tu terminal, usará por defecto el logo que seleccionaste.
